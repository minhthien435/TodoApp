# TodoApp
- A sleek and lightweight TodoList application designed to help you organize your work.
---
## Core features
-Built to meet the following requirements:
**Task : View all your pending and completed tasks 
* **Full CRUD Operations**
* **Create:** Add new tasks to your list
* **Read** Task are fetched and displayed instantly
* **Update** Remove tasks that no longer relevant
* **Status Toggling** Simple to click on a task to to toggle between "Completed" and "Pending" status
* **Smart filtering** Filter all tasks base on status : *All* *Active* or *Completed*
* **Data Persistance** Using **Local Storage** to ensure the data is saved even after refreshing the page or closing the browser
---
## Tech Stack
This project is built based on:
1. **HTML5** Structured the user interface.
2. **CSS3** Styled the application for a modern and responsive look.
3. **Javascript** Handled the logic,state,and storange integration.
